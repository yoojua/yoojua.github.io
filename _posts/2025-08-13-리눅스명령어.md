# 리눅스 기본 명령어 정리

![](https://velog.velcdn.com/images/mymy/post/757070a5-699e-4c92-af69-ab174171d716/image.png)


---

## ① `ls`

**LiSt**의 약자로 Windows 명령 프롬프트의 `dir` 명령과 같은 역할을 합니다.
현재 디렉터리(폴더)에 있는 파일 목록을 나열합니다.

**\[사용 예]**

```bash
ls                                   # 현재 디렉터리의 파일 목록 표시
ls /etc/sysconfig /etc/sysconfig    # 디렉터리의 목록 표시
ls -a                                # 숨김 파일 포함 목록 표시
ls -l                                # 자세히 표시
ls *.cfg                             # 확장자가 cfg인 파일 목록
ls -l /etc/sysconfig/a*              # 앞 글자가 'a'인 파일의 상세 목록
```

---

## ② `cd`

**Change Directory**의 약자로 디렉터리를 이동합니다.

**\[사용 예]**

```bash
cd                       # 홈 디렉터리로 이동
cd ~rocky rocky          # 사용자의 홈 디렉터리로 이동
cd ..                    # 상위 디렉터리로 이동
cd /etc/sysconfig        # 절대 경로로 이동
cd ../etc/sysconfig      # 상대 경로로 이동
```

**`.` 와 `..`**

* `.` : 현재 디렉터리
* `..` : 상위 디렉터리
  예: `cp ../lib/abc .` → 상위 디렉터리의 `/lib/abc`를 현재 디렉터리(`.`)로 복사

---

## ③ `pwd`

**Print Working Directory** — 현재 디렉터리의 전체 경로를 표시합니다.

**\[사용 예]**

```bash
pwd    # 현재 작업 중인 디렉터리 경로 출력
```

**✅ 숨김 파일**

* 파일/디렉터리 이름 앞에 `.`을 붙이면 자동으로 숨김 파일이 됨

---

## ④ `rm`

**ReMove** — 파일 또는 디렉터리 삭제.

**\[사용 예]**

```bash
rm abc.txt       # 파일 삭제
rm -i abc.txt    # 삭제 확인 후 삭제
rm -f abc.txt    # 강제 삭제
rm -r abc        # 디렉터리 삭제
rm -rf abc       # 하위 디렉터리 포함 강제 삭제
```

---

## ⑤ `cp`

**CoPy** — 파일/디렉터리를 복사합니다.

**\[사용 예]**

```bash
cp abc.txt cba.txt     # 이름 변경하여 복사
cp -r abc cda          # 디렉터리 복사
```

---

## ⑥ `touch`

크기가 0인 새 파일을 생성하거나, 파일의 최종 수정 시간을 변경.

**\[사용 예]**

```bash
touch abc.txt    # 없으면 새로 생성, 있으면 수정 시간 변경
```

---

## ⑦ `mv`

**MoVe** — 파일/디렉터리 이동 또는 이름 변경.

**\[사용 예]**

```bash
mv abc.txt /etc/sysconfig/     # 디렉터리로 이동
mv aaa bbb ccc ddd            # 여러 파일 이동
mv abc.txt www.txt            # 이름 변경
```

---

## ⑧ `mkdir`

**MaKe DIRectory** — 새 디렉터리 생성.

**\[사용 예]**

```bash
mkdir abc            # 디렉터리 생성
mkdir -p /def/fgh    # 상위 디렉터리까지 자동 생성
```

---

## ⑨ `rmdir`

**ReMove DIRectory** — 비어있는 디렉터리 삭제.

**\[사용 예]**

```bash
rmdir abc    # 디렉터리 삭제
```

---

## ⑩ `cat`

**conCATenate** — 파일 내용을 화면에 출력.

**\[사용 예]**

```bash
cat a.txt    # 파일 내용 출력
```

---

## ⑪ `head`, `tail`

텍스트 파일의 앞/뒤 일부 출력.

**\[사용 예]**

```bash
head anaconda-ks.cfg       # 앞 10행
head -3 anaconda-ks.cfg    # 앞 3행
tail -5 anaconda-ks.cfg    # 마지막 5행
```

---

## ⑫ `more`

페이지 단위로 텍스트 파일 출력.

* `[Space]`: 다음 페이지
* `[B]`: 이전 페이지
* `[Q]`: 종료

**\[사용 예]**

```bash
more anaconda-ks.cfg
more +30 anaconda-ks.cfg    # 30행부터 출력
```

---

## ⑬ `less`

`more`와 유사하지만 기능 확장.

* 화살표 키, `[PageUp]`, `[PageDown]` 사용 가능

**\[사용 예]**

```bash
less anaconda-ks.cfg
less +30 anaconda-ks.cfg
```

---

## ⑭ `file`

파일의 종류 표시.

**\[사용 예]**

```bash
file anaconda-ks.cfg    # ASCII text
file /dev/sr0           # block special
```

---

## ⑮ `clear`

터미널 화면을 깨끗하게 지움.

**\[사용 예]**

```bash
clear
```
